# Niri Dotfiles

## Features

- **Niri Window Manager** - A modern Wayland tiling window manager
- **Custom Configuration** - Pre-configured Niri layout, bindings, and appearance
- **Keyboard Layout** - VMK (Vietnamese Keyboard) support integrated with Fcitx5
- **Theme & Appearance** - Minegrub GRUB theme and SDDM Astronaut theme
- **Essential Tools** - Pre-configured with git, browsers, media players, and system utilities
- **Font Support** - Full font support including Noto fonts, CJK, and Delugia Code

## Installation

Run the install script to set up the entire environment:

```bash
curl -fsSL https://raw.githubusercontent.com/taitapcode/dotfiles/niri/install.sh | bash
```
