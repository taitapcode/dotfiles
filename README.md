# Hyprland Dotfiles

Hyprland window manager setup for Arch-based systems with automatic installation.

## Quick Start

```bash
curl -fsSL https://raw.githubusercontent.com/taitapcode/dotfiles/hyprland/install.sh | bash
reboot
```

## What's Included

- Hyprland window manager with custom keybindings
- HyprPanel status bar and dashboard
- Rofi application launcher
- Swaync notifications
- Fcitx5 Vietnamese input method
- Catppuccin Mocha theme
- Wallpaper management with swww
- System utilities (screenshots, brightness control, etc.)

## Post-Installation

```bash
nwg-look           # Configure theme
fcitx5-configtool  # Set input method
```
